# Sorting Visualizer

An interactive web-based **Sorting Visualizer** built using **HTML, CSS, and JavaScript** to help understand how popular sorting algorithms work internally.  
The application visually demonstrates the step-by-step execution of sorting algorithms using animated bars whose heights represent array values.

## âš™ï¸ How It Works

- Each bar represents an element of the array.
- The height of a bar corresponds to the value of that element.
- When a sorting algorithm is selected, the algorithm runs step-by-step.
- During sorting:
  - Bars are compared and swapped visually.
  - Animation speed is controlled by the speed slider.
- A new random array can be generated at any time using the **New Array** button.
